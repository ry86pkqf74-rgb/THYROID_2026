CLASSIFICATION TASK
-------------------
Answer each question strictly based on the TEXT SNIPPET above.

1. IS THIS A TRUE ETE EVENT?
   - TRUE: The note is DOCUMENTING an actual extrathyroidal extension finding
   - FALSE: The note is discussing ETE as a risk, surgical risk disclosure, consent language,
     or educational mention without documenting that the patient HAS it

   FALSE POSITIVE SIGNALS (common patterns):
   - "risks include ... extrathyroidal extension"
   - "risk of extrathyroidal spread was discussed"
   - Consent language listing surgical risks
   - Preoperative discussion of possible ETE before pathology is known
   - Imaging mentioning "cannot exclude" or "possible" ETE (mark as SUSPECTED, not confirmed)

   TRUE POSITIVE SIGNALS:
   - Pathology report: "extrathyroidal extension present, microscopic/gross"
   - Operative note: "tumor was found to extend beyond the thyroid capsule"
   - Operative note: "gross extrathyroidal extension noted intraoperatively"
   - Discharge summary stating the final pathology diagnosis includes ETE
   - Endocrine note reviewing pathology results that document ETE

2. ETE GRADE (if is_true_ete = true or suspected):
   - "gross": extends into strap muscles, trachea, esophagus, RLN, larynx, mediastinum;
              "pT4", "T4b", "extensive", "macroscopic", "invades trachea/esophagus/RLN"
   - "microscopic": confined to perithyroidal soft tissue only, minimal fat invasion;
                    "minimal", "microscopic", "focal", "perithyroidal fat only", "pT3b"
   - "present_ungraded": ETE documented but grade not specified
   - "suspected": imaging only or "cannot exclude" language
   - "none": explicitly states no ETE

3. SOURCE CATEGORY:
   - "path_report": formal pathology report, synoptic report, surgical pathology
   - "op_note": operative note documenting intraoperative findings
   - "ct_report": CT scan radiology report
   - "us_report": ultrasound radiology report
   - "discharge": discharge summary
   - "endocrine": endocrine clinic follow-up note
   - "h_p_consent": H&P note or consent discussion (boilerplate risk language)
   - "other": other note types

4. PATHOLOGIC CONFIRMATION:
   - true: from a formal pathology report (gold standard)
   - false: from clinical observation, imaging, or other non-pathologic source
